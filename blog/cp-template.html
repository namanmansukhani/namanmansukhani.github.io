A great started C++ competitive programming template

A good template for competitve programming can not just cut down on coding time for problems, it can also greatly reduce execution time. I’m going to walk you through a competitive programming template. It can be found on my Github.

Commands to quickly create and run files

Script run compiles a file with g++, and runs the created executable a.out, which takes input from input.txt.

./run file_name

Here, run calls the make command, which ensures that the file is not compiled unnecessarily. The generated executable is only run if the make command was successful.

Script new copies the template into a new file.

./new file_name

Here, new creates a new file file_name with the contents of template.cpp

Explaining the template

The first step is to use type int as type long long. In order to make this work, we need to first change int main() to signed main() since the type of the main function cannot be long long.

Then, we add these lines for conveniently writing loops.

#define rep(i,n) for (int i = 0;i < n;i++)
#define REP(i,k,n) for (int i = k;i < n;i++)

So, something like for (int i=0;i<n;i++) becomes rep(i,n), which is more concise.

We also add - cpp ios_base::sync_with_stdio(false);cin.tie(NULL);cout.tie(NULL);

at the start of the main function.

This makes sure that C I/O functions like printf and scanf don’t have to be synced with the C++ I/O functions like cin and cout. This boosts I/O performance for cin and cout. It is honestly surprising to see the difference that this optimization makes.
