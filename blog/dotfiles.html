Storing my dotfiles on Github

Whenever I setup a new server or changed my machine, moving dotfiles was a hassle. So I decided to archive them on github.

My setup

I created a new directory ~/dotfiles/, and I stored my dotfiles here. Then, I created a hard link between the files here and the files in ~/. So, whenever I make changes to the files and pull those changes from Github, the changes are propagated to the dotfiles in ~/.

#!/usr/bin/env bash
for file in .vimrc .zshrc .tmux.conf .bashrc
do
    ln -f ~/dotfiles/$file ~/$file
done

Installation

Simply clone the repo and run the install.sh script like so:

git clone https://github.com/namanmansukhani/dotfiles.git
cd ~/dotfiles
./install.sh

Now, all the dotfiles are present and can be easily moved betweeen machines!
